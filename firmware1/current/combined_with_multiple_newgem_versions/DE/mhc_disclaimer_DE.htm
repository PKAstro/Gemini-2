Dieses Dienstprogramm ist für den Einsatz mit einem Android Phone/Tablet oder iphone/ipad Klasse des Geräts. Der obige Link führt Sie zu den mobilen Controller. Um dieses Dienstprogramm zu verwenden ist eine WLAN-Verbindung für das Telefon/Tablet und das Gemini-2 angeschlossen werden auf dem gleichen Netzwerk. Diese Software wurde von einem Benutzer der Gemini-2 und ist ein Beispiel von, was ein Benutzer tun kann mit dem Open Source Web Interface.

Klicken Sie auf das Mobile Hand controller link oben zum Ausführen dieses Dienstprogramms.

Klicken Sie auf Startseite, um zum Hauptmenü zurückzukehren
