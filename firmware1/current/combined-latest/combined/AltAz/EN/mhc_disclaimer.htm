MHC : [ Mobile Hand Controller]
[ HOME PAGE ]

This Utility is for use with an Android Phone/Tablet or Iphone/Ipad class of device. The above link takes you to the Mobile Hand Controller. To use this utility requires an WiFi link for the phone/tablet and the Gemini-2 be connected on the same network. This software was supplied by a user of the Gemini-2 and is an example of what a user can do with the open source Web interface.

Click on the Mobile Hand controller link above to run this utility.  If using a Cell Phone or tablet, you will more than likely need to use your Gemini-2's IP address.  Example http://192.168.0.111/MHC/mhc.html

Click on HOME PAGE to return to main page.
