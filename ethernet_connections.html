    • The top three illustrations are for direct connections. If either the router or switch has a Gigabit (10/100/100) speed connection it does not matter where a straight or cross-over cable is used.

      Such adapters are described in FAQ Question  #3  There are many brands of what is commonly called Gateway/Bridges.  Just make sure that it has a Bridge mode.

Options 1 through 4


  1. Option 5 illustrates how you can use a single USB output from you computer to connect your cameras and also the Gemini-2 using it's Ethernet port. It adds a USB to Ethernet adapter into the mix so that you have the best of both worlds with only one USB cable from your computer.

    PLEASE NOTE: If you are using a USB to ETHERNET ADAPTER, make sure you go into Windows Advanced Power setting and make sure "USB selective suspend setting" is set to disabled.  See image at bottom of page.

    • This requires the remote computer to have two hardwired Ethernet ports.  Most computer so not have a second hardware Ethernet port. Option 6


  1. Option 7 illustrates how you use computers two connected via a Wifi network, (maybe using teamview or Remote desktop) to control the Gemini-2 plus camera and focuser. 
    • This requires the remote computer to have one hardwired Ethernet port and one Wifi port.  Most laptops fall into this category.
    • The Ethernet cable that connects the remote computer to the Gemini-2 can be a normal Ethernet cable if the Ethernet port on the laptop is rated as a Gigabit speed port.
    • It the Ethernet port on the remote computer is only a 10/100MHz speed Ethernet port then an Ethernet crossover cable is required between the remote computer and the Gemini-2.
