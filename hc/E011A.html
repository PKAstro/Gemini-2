Interactive Hand-Controller Menus - click on green button to Navigate
This is the screen where you enter your local time. Enter the time in 24-hour format. The format is Hours:Minutes:Seconds

Enter time 30 seconds in the future, as that will give you enough time to get back to the time / date screen (click Ent to go back) and then to click the Set button on this screen to set the correct time.

NOTE: If you only use the UTC time by setting the timezone offset to zero, then enter the correct UTC time in this screen.
