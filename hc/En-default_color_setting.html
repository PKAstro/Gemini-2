This is the default Color setting on the Gemini-2
How the color setting react in Gemini-2 hand controller
Adaptive Color Checked* Startup Daytime Startup Sunset Startup Night
Daytime (Green) mode selected Green Green Green
Dawn (Blue) mode selected Green Blue Blue
Nighttime (Red) mode selected Green Blue Red
Adaptive color  Not Selected      
Daytime (Green) mode selected Green Green Green
Dawn (Blue) mode selected Blue Blue Blue
Nighttime (Red) mode selected Red Red Red
*If Adaptive color is selected + Daytime Mode (Green) is selected, this is the same as having the Adaptive mode turned off  and green selected.  The screen will always be green unless you manually change it.

*If Adaptive color is selected + Dawn  Mode (Blue) Selected, and Gemini is started up before sunset, it will startup green, then it will turn blue at shortly after sunset and remain blue all night. But to get it to change you either have to unplug the hand controller, and plug it back in or turn power off and back on.  It does not automatically change.  Note: unplugging the hand controller and plugging it back in is the preferred method.

*If Adaptive color is selected + Nighttime Mode (Red) Selected, and Gemini is started up before sunset, it will start up green, then 10° after sunset, it will turn blue and, and then in another 10° red.   So if you want it to go  through all 3 colors changes then select Adaptive Color and Red.  But to get it to change you either have to unplug the hand controller, and plug it back in or turn power off and back on.  It does not automatically change.  Note: unplugging the hand controller and plugging it back in is the preferred method.

Also, once you hit the back button, after selecting a color and mode, on the screen that comes up is a "save setting" button on the bottom left.  You have to hit that for the setting to stay through power cycle.

Also note that Dim setting is saved between colors.  So before you turn off your mount at night with a Red Dimmed display, hit the bright button utill it is at it's brightest level.  This will insure that when it's turned on, that you will be able to see the screen.
