Interactive Hand-Controller Menus - click on green button to Navigate
This screen contains the Language, Display, SD Card, Buttons, Mode and Info selection sub-menus.

We will do each menu in the order above.

Clicking on one of the button on the left will take you to that menu.  I suggest you try the Language button first, to see how it works.

The back button will take you to the previous menu.

The Store Settings button stores all the settings changes you made from the sub-menus of this menu. You must select this to store the setting
