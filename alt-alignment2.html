Adding Stars to the Model - Manual method. 
This page show the steps to add one star to the Model and look at the parameters
using the manual method.  There is also a semi-automatic method that you can use.
  5. use the arrow keys to center target and then
     hit Menu, to go and see the modeling
     parameters.


    To just add another star without
    looking at the parameters, hit Goto and then
    Goto Bright Star at the bottom of that page


  6. Now select Align


 


Hit Alignment to add the star to the Alignment.









   7. To add another star to the model you would select "GoTo Bright Star.  However we want to see the Model Parameters.  Note this is an optional step. 

So Select Model Parameters Please.


Notice that you only have only one or 2 entries for the Error Index.  Only the HA Index Error and Dec Index Error is calculated on the first star.

You have just added your first star to the West model.

You use the BACK button to take you to the next Star selection screen.

The Switch button will switch the Error screens between the East and West model Error screens.

The UNDO button will remove the Last star added to the model.

Please hit Back for this Demo

This screen shows the mount alignment parameters that the current model has.

The math behind calculating these errors:
1st alignments set the index parameters ID, IH.

3-5 alignments calculate MA, ME additionally.

6th..9th alignment add CH and NP non-perpendicularity parameters to the model.

From 10th alignment up counterweight flexure DAF is added.

All Values are in arc-seconds.


  This ends this Demo for adding one star to the model.

  If you were doing this on your mount, you would Goto Bright Star and add more stars, and repeat these steps with a different stars, until you added at least 3 stars on each side of the meridian.  You can also get to the Bright star menu from the main page. Goto-->Bright Star and select the star from the list.
