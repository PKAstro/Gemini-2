Connecting Gemini 2 to Computers and other devices

  1. Stand-alone: Note that the Gemini-2 can be used completely stand alone, without the need of an external computer..  Here is a tutorial of the hand controller.
  2. Serial Port connector: You can use the Gemini-2 Serial port to connect to a PC that has a Serial port or a USB to serial adapter. For the cable wiring see Serial Port Wiring.  The Gemini-2 port is labeled "Serial Port 1" on the Original Gemini-2 and "Serial/GPS" on the Gemini-2 Mini.
    1. Please note that you cannot access the web interface using the Serial Port.
    2. The serial port only accepts the Gemini commands listed here.
    3. The serial port can be used with the ASCOM driver.  See 4c below.
  3. USB connector:  This requires a USB driver for Windows.  There is no driver for MAC. Linux computers have a driver built in.
    1. The USB-A connector can only supply +5V, and is not supported otherwise.
    2. Please note that you cannot access the web interface using the USB Port.
    3. The USB port only accepts the Gemini commands listed here.
    4. The USB port can be used with the ASCOM driver.  See 4c below.
  4. Ethernet connector:  The Ethernet connector supports a Web interface that can control just about every function of the Gemini-2.  When the Gemini-2 is connected to your local network using the Ethernet connector, any device with access to your local network should be able to connect to the Gemini-2.  This includes Windows computers, MAC computers, Linux computers, iPhones and iPads, Android phones and Android tablets. 
    1. Instructions to connect the Ethernet interface to your Windows computer are here.
    2. To access the Web Interface see Web Interface instructions.
    3. To use any program that supports the ASCOM Standard, you need to load the ASCOM Standard and the ASCOM Gemini Driver.  See instructions here.
    4. To configure the ASCOM driver to use the Ethernet port see Configuring the Ethernet connection in the ASCOM Driver.   Note: The Ethernet connection must be working.  If you cannot connect to the Web Interface, you will not be able to connect to the ASCOM driver using the Ethernet port.
    5. There are no instructions to connect the Ethernet interface to a MAC or Unit computer. You can access it by going to the IP address shown under Menu→Mount→Network in the hand controller.
    6. It is possible to use an Travel Router or Bridge to add WiFi to the Gemini-2 Ethernet port. See info.
