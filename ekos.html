Ekos Automation Tool for Astronomy

Ekos is an advanced cross-platform (Windows, OSX, Linux) observatory control and automation tool with particular focus on Astrophotography.

Ekos works as a client to an INDI server, through which the various pieces of equipment are controlled. There is a wide range of possible implementations of this, for example:

  • Running Ekos and the INDI server on the same computer, to which the Gemini and cameras are connected
  • Running Ekos and the INDI server on separate computers, allowing for remote control of the telescope

    • Often the INDI server is run on a small one-board device, like a Raspberry Pi at the telescope.
